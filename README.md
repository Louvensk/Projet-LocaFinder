# LweKay
Trouver un logement à louer est devenu un casse-tête dans notre région, c’est pourquoi Loca_Finder a été conçu pour aider les gens à trouver facilement un logement. L’idée est de faciliter la mise en location des maisons pour les propriétaires et d’éviter aux locataires de trop chercher dans les rues pour trouver un logement à louer.

# Must have
- [x] Les proprietaires peuvent creer un compte utilisateur afin d'acceder a l'application.
- [x] Les propriétaires peuvent indiquer le prix, leur numéro de téléphone et la localisation de leur logement
- [x] L'utilisateur qui souhaite trouver un logement peut acceder a l'application avec ou sans compte utilisateur.
- [x] L'utilisateur peut chercher un logement n'importe où dans le pays.
- [x] L'utilisateur peut consulter la liste des logements à louer dans une zone spécifique.
- [x] L'utilisateur peut voir le prix d'un logement.
- [x] L'utilisateur peut contacter le propriétaire directement via le numéro de téléphone indiqué.
- [x] L'utilisateur peut se deconnecter de l'application.
